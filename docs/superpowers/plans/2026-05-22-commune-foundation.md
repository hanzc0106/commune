# Commune Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the initial monorepo foundation for Commune with a Go API, PostgreSQL, React/Vite frontend shell, Docker Compose, and repeatable local development commands.

**Architecture:** The repository is a lightweight monorepo with `apps/api` for the Go backend and `apps/web` for the React frontend. The Go API owns configuration, database connectivity, health checks, and static asset serving; the React app is built into static files and served by the API in production. PostgreSQL runs through Docker Compose for local development and deployment.

**Tech Stack:** Go, Chi, pgx, sqlc, PostgreSQL, React, Vite, TypeScript, Tailwind CSS, Radix UI, Docker Compose, PowerShell scripts.

---

## Scope

This plan creates the foundation only. It does not implement authentication, initialization, members, categories, transactions, budgets, or the final mobile UI.

The completed foundation must provide:

- A clean monorepo layout.
- A Go API that starts locally.
- A `/healthz` endpoint.
- PostgreSQL available through Docker Compose.
- A verified Go database connection path.
- sqlc configured and runnable.
- A React/Vite frontend shell.
- A production path where the Go API can serve frontend static files.
- PowerShell scripts for local development, build, and migration commands.
- Basic documentation and environment examples.

import "os"

type Config struct {
	HTTPAddr    string
	DatabaseURL string
	StaticDir   string
}

func Load() Config {
	return Config{
		HTTPAddr:    getEnv("COMMUNE_HTTP_ADDR", ":8080"),
		DatabaseURL: getEnv("COMMUNE_DATABASE_URL", "postgres://commune:commune@localhost:5432/commune?sslmode=disable"),

}

func getEnv(key string, fallback string) string {
	value := os.Getenv(key)
	if value == "" {
		return fallback
	}
	return value
}

import (
	stdhttp "net/http"

	"github.com/go-chi/chi/v5"
)

func NewHandler() stdhttp.Handler {
	r := chi.NewRouter()

	r.Get("/healthz", func(w stdhttp.ResponseWriter, r *stdhttp.Request) {
		w.Header().Set("Content-Type", "text/plain; charset=utf-8")
		w.WriteHeader(stdhttp.StatusOK)

	})

	return r
}

package db

import (
	"context"
	"errors"

	"github.com/jackc/pgx/v5/pgxpool"
)

func Open(ctx context.Context, databaseURL string) (*pgxpool.Pool, error) {
	if databaseURL == "" {
		return nil, errors.New("database URL is required")
	}

	pool, err := pgxpool.New(ctx, databaseURL)
	if err != nil {
		return nil, err
	}

	if err := pool.Ping(ctx); err != nil {
		pool.Close()
		return nil, err
	}

	return pool, nil
}
```

package http

import (
	stdhttp "net/http"
	"os"
	"path/filepath"

	"github.com/go-chi/chi/v5"
)

type Options struct {
	StaticDir string
}

func NewHandler(options Options) stdhttp.Handler {
	r := chi.NewRouter()

	r.Get("/healthz", func(w stdhttp.ResponseWriter, r *stdhttp.Request) {
		w.Header().Set("Content-Type", "text/plain; charset=utf-8")
		w.WriteHeader(stdhttp.StatusOK)
		_, _ = w.Write([]byte("ok\n"))
	})

	if options.StaticDir != "" {
		fileServer := stdhttp.FileServer(stdhttp.Dir(options.StaticDir))
		r.NotFound(func(w stdhttp.ResponseWriter, r *stdhttp.Request) {
			path := filepath.Join(options.StaticDir, filepath.Clean(r.URL.Path))
			if info, err := os.Stat(path); err == nil && !info.IsDir() {
				fileServer.ServeHTTP(w, r)
				return
			}
			stdhttp.ServeFile(w, r, filepath.Join(options.StaticDir, "index.html"))
		})
	}

	return r
}
```
